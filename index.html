Notable Changes This version uses the 3.2.0 kafka-clients version Non-blocking retry bootstrapping is now more robust New Error Handler Mode By default, after an error, the DefaultErrorHandler performs seeks on the remaining records from the last poll and re-fetches them from the broker on the next poll.
