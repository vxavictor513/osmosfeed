This enhancement brings a significant improvement to how default StorageClasses are assigned to PersistentVolumeClaims (PVCs). With this feature enabled, you no longer need to create a default StorageClass first and then a PVC to assign the class. How to use it? As this feature has graduated to GA, there's no need to enable a feature gate anymore.
